# Terminal Calculator in C
A robust command-line calculator built to practice C fundamentals, including control flow, switch cases, and mathematical functions.

## Features
* **Basic Arithmetic:** Addition, Subtraction, Multiplication, Division.
* **Advanced Operations:** Power ($x^y$) and Remainder (fmod).
* **Error Handling:** Prevents crashes from division by zero and invalid non-numeric inputs.
* **Continuous Loop:** Perform multiple calculations without restarting the program.

## How to Run
1. Clone the repo: `git clone https://github.com/amanbhardwaz20/C-Terminal-Calculator.git`
2. Compile: `gcc Calculator.c -lm -o calculator`
3. Run: `./calculator`
